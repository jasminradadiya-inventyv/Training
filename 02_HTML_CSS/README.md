# 02_HTML_CSS

A clean and responsive **Login & Registration Page** building used HTML and CSS.

## Overview
This project demonstrates a split-screen layout with a tabbed form interface for users to sign in or create an account for "ABC INDIA PVT. LTD."

## Features
- **Tabbed Interface**: Pure HTML/CSS toggle between Login and Register forms.
- **Responsive Design**: Adapts for different screen sizes.
- **Form Elements**: Includes inputs for username, password, and email with validation attributes.

## Structure
- `index.html`: Main structure of the page.
- `style.css`: Styling regarding layout, colors, and the tab switching mechanism.
- `logo.jpeg`: Project asset.
